Introduction

JavaScript is a versatile programming language used for web development.

It allows for interactive features, dynamic content, and more.

Variables

Variables are containers for storing data values.

let name = "Alice";
const PI = 3.14;

Functions

Functions are blocks of code designed to perform tasks.

function greet(name) {
  return "Hello, " + name;
}

Loops

Loops are used to repeat a block of code.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Objects

Objects store key-value pairs and represent complex data.

let person = {
  name: "Bob",
  age: 30
};
